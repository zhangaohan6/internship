### 跨境电商选品 + 利润 + 广告分析工具　|　个人项目　|　Python
*GitHub: internship/ecom-product-selector*

- 独立开发**选品与利润分析引擎**：自动计算 FBA 全链路**利润瀑布**（售价 − 平台佣金 − FBA 费 − 仓储 − 采购成本 − 头程 − 广告 → **净利 / 毛利率 / ROI / 盈亏平衡 ACoS**），把运营手工 Excel 测算自动化。
- 设计 **0–100 选品机会评分模型**（需求 × 利润 × 评论壁垒 × 竞争度，带最低毛利硬门槛），对候选产品自动排序出 shortlist。
- **在 1001 个真实 Amazon 产品数据上验证**：发现仅 **376 个（38%）** 在扣除佣金 + FBA + 广告等全链路费用后能过 10% 毛利门槛——量化揭示"很多产品（尤其电子类）单位经济模型其实不成立"。
- 开发 **PPC 关键词分析模块**：逐词计算 ACoS / ROAS / **最大可盈利 CPC / 建议出价**，自动标出哪些词可盈利、值得投。
- 工程化交付：可直接导入 **Helium 10 / Jungle Scout** 真实数据；含命令行工具、**Streamlit 可视化仪表盘**、8 个单元测试，零依赖、克隆即跑。

### 销量预测 + 库存补货计划工具　|　个人项目　|　Python
*GitHub: internship/ecom-restock-planner*

- 独立开发 **FBA 补货决策工具**，解决"何时补货、补多少"这一卖家头号痛点（断货掉排名 / 压货占资金）。
- 实现 **4 种销量预测方法**（移动平均 / 指数平滑 / Holt 趋势 / 周季节性），并用**回测（MAPE）为每个 SKU 自动选最优模型**。
- 计算完整补货决策：**安全库存、再订货点、可售天数、断货风险、按最小起订量取整的补货量**，并按紧急度排序、给出建议采购金额。
- **在真实零售数据集上验证**（UCI Online Retail，**54 万笔交易 / 4070 个商品**）：前 40 个商品中 **23 个需立即补货、最快 15 天断货、建议采购额 ≈ $45k**。
- **发现真实需求高度间歇/爆发性**（大量零销量天），简单预测器在稀疏 SKU 上 MAPE 偏高——据此提出引入 **Croston 间歇需求模型** 作为改进方向。
- 工程化交付：命令行 + **Streamlit 仪表盘**（预测曲线 + 补货计划）+ 8 个单元测试，支持导入真实销量数据。

### 多平台广告 ROAS 优化分析器　|　个人项目　|　Python · pandas
*GitHub: internship/ecom-ads-optimizer*

- 独立开发**广告效果分析器**：吃进 Google / Meta / TikTok 多平台广告数据，逐 campaign 计算 **ROAS / ACoS / CPA / CTR / CVR**，并按**盈亏平衡 ROAS**（= 1 / 毛利率）识别在亏损的广告花费。
- **在真实结构的多平台广告数据集上验证**（1800 行 campaign 数据，$1110 万广告花费 / $5418 万收入）：发现 **48% 的广告花费（$540 万）低于盈亏平衡、在赔钱**；且 **Google Ads 占 57% 预算却 ROAS 最低（3.47），而 TikTok（ROAS 7.62）预算不足**——据此给出**预算再分配建议**。
- 工程化交付：CLI + **Streamlit 仪表盘** + 6 个单元测试。
